# Imagen — Image Generation

> Google's image generation models. From quick prototypes to 4K production assets.

---

## Models Overview

| Model | Best For | Resolution | Speed |
|-------|----------|------------|-------|
| **Imagen 4** | Production, quality | Up to 4K | Slower |
| **Imagen 3** | General use | 1024px | Medium |

---

## Imagen 4

The flagship image generation model.

### Features
- **4K resolution** output
- **Text rendering** — legible text in images
- **Reasoning-enhanced** composition
- **Google Search grounding** — real-world accuracy
- **"Thinking" process** — refines composition before generating

### Model String
```python
"imagen-4-generate"
```

### Code Example

```python
from google import genai

client = genai.Client()

response = client.models.generate_images(
    model="imagen-4-generate",
    prompt="A futuristic Tokyo street at night, neon signs with Japanese text, rain-slicked streets, cinematic lighting",
    config={
        "number_of_images": 4,
        "aspect_ratio": "16:9"
    }
)

for i, image in enumerate(response.generated_images):
    image.save(f"image_{i}.png")
```

## Nano Banana Pro (Gemini 3 Pro Image)

Fast image generation with character consistency.

### Features
- **Character consistency** — same character across images
- **Multi-turn editing** — conversational refinement
- **Fast generation** — optimized for speed
- **World knowledge** — understands real concepts

### Model String
```python
"gemini-3-pro-image"  # Official name
# Also known as "Nano Banana Pro"
```

### Code Example

```python
# Generate with character consistency
response = client.models.generate_images(
    model="gemini-3-pro-image",
    prompt="A cartoon fox character wearing a blue scarf, standing in a forest",
    config={"number_of_images": 1}
)

# Continue with same character
response2 = client.models.generate_images(
    model="gemini-3-pro-image",
    prompt="The same fox character now sitting by a campfire",
    config={
        "reference_images": [response.generated_images[0]],
        "number_of_images": 1
    }
)
```

## Safety & Watermarking

All images generated by Google's models include:

- **SynthID** — Invisible digital watermark
- **Safety filters** — Content moderation
- **Visible watermark** — Optional, configurable

```python
# Control person generation
config = {
    "person_generation": "allow_adult",  # or "dont_allow"
    "safety_filter_level": "block_medium_and_above"
}
```

## Best Practices

### 1. Be Specific

```python
# ❌ Vague
"A dog"

# ✅ Specific
"A golden retriever puppy sitting in autumn leaves, soft natural lighting, shallow depth of field, professional pet photography"
```

### 2. Include Style

```python
# Specify artistic style
"...in the style of Studio Ghibli animation"
"...photorealistic, 8K, cinematic"
"...watercolor illustration, soft edges"
"...3D render, octane render, volumetric lighting"
```

### 3. Negative Prompts

```python
config = {
    "negative_prompt": "blurry, low quality, distorted, watermark"
}
```

## Comparison with Gemini Image

| Aspect | Imagen | Gemini Image (Nano Banana) |
